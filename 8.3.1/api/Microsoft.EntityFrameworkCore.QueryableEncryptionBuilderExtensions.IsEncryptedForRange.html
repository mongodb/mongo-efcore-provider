IsEncryptedForRange<T>(PropertyBuilder<T>, T, T, Guid, Action<QueryableEncryptionRangeBuilder<T>>?)

Configures a property to be encrypted using MongoDB Queryable Encryption with support for range queries.

public static PropertyBuilder<T> IsEncryptedForRange<T>(this PropertyBuilder<T> builder, T minValue, T maxValue, Guid dataKeyId, Action<QueryableEncryptionRangeBuilder<T>>? rangeBuilder = null)

Parameters

builder PropertyBuilder<T>

The PropertyBuilder for the property being configured.

minValue T

The minimum permitted value for this range.

maxValue T

The maximum permitted value for this range.

dataKeyId Guid

The encryption data key id to be used as a Guid.

rangeBuilder Action<QueryableEncryptionRangeBuilder<T>>

The additional configuration for configuring range options.

Returns

PropertyBuilder<T>

The same builder instance so that multiple calls can be chained.

Type Parameters

T

The type of property being configured.

Remarks

Only MongoDB.Bson.BsonType.Decimal128, MongoDB.Bson.BsonType.Double, MongoDB.Bson.BsonType.Int32, and MongoDB.Bson.BsonType.Int64 storage types are permitted for range encryption.
